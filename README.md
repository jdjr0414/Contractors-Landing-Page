# Contractor Working Capital Starter

SEO-first Astro starter for an industry-specific lead-gen site that funnels users to Axiant Partners.

## Site goal
Generate leads from contractors searching for:
- contractor working capital
- construction business funding
- contractor cash flow problems
- construction equipment financing
- payroll gaps between jobs

## Recommended first steps in Cursor
1. Open this folder.
2. Run `npm install`
3. Run `npm run dev`
4. Ask Cursor to:
   - refine design
   - expand content
   - add 10 contractor blog posts
   - verify metadata and canonicals
   - connect CTA links to your preferred Axiant page

## Core routes included
- `/`
- `/contractor-working-capital`
- `/construction-equipment-financing`
- `/contractor-line-of-credit`
- `/blog`
- sample blog posts

## Content model
- Core pages live in `src/content/pages`
- Blog posts live in `src/content/blog`
- Shared site config lives in `src/data/site.ts`

## Important
This starter is set up for static SEO pages with route-specific metadata.
